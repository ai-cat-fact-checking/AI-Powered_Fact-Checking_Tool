# Stage3 疑美論文章分類與標註流程（API 規則說明）

ref: https://gitmind.com/app/docs/mo2ke1ve

## Step 1：檢查文章是否提及「台美關係」

- 若文章中至少提及一次（包含一次）台美關係 → 進入 **Step 2**。
- 若沒有 → 標註為「非判斷對象」。

---

## Step 2：判斷文章情緒傾向

- 若文章帶有**負面語意** → 進入 **Step 3**。
- 若文章未帶有負面語意 → 標註為「非判斷對象」。

---

## Step 3：區分「事實」與「觀點」

1. **事實檢查**
   - **資訊來源**
     - 是否來自中國 → Tag: `來源:中國`

   - **事實內容**
     - 是否「有誤」 → Tag: `事實有誤`
     - 是否「完整」 → Tag: `事實不完整`

2. **觀點檢查**
   - 驗證文章由「事實」到「觀點」的推論是否合理，若不合理，標註以下謬誤：
     - 滑坡謬誤 → Tag: `謬誤:滑坡`
     - 假兩難謬誤 → Tag: `謬誤:假兩難`
     - 偷換概念 → Tag: `謬誤:偷換概念`
     - 不當類比 → Tag: `謬誤:不當類比`

3. **情緒分析（NLP）**
   - 是否包含**個人情緒字眼** → Tag: `情緒化`

---

## Step 4：判斷是否為「資訊操作」

- 規則：若 **Step 3** 中任一檢查結果為「是」，則判定文章可能屬於「資訊操作」。
  - Tag: `資訊操作`

---

## Step 5：資訊操作文章的細分類

若文章被判定為「資訊操作」，則根據 **Key words** 與論述內容進一步加上分類標籤（可多選）：

1. **棄子論**
   - Keyword: 美國把台灣當作棋子，最終會拋棄台灣
   - Tag: `棄子論`

2. **實力衰弱**
   - Keyword: 美國實力衰弱，無法保護台灣
   - Tag: `衰弱論`

3. **世界戰亂的禍源**
   - Keyword: 美國是世界戰亂的根源
   - Tag: `亂源論`

4. **無實質效益**
   - Keyword: 美國宣稱支持台灣卻無實質幫助，反而讓台灣受害，從中牟利
   - Tag: `假朋友`

5. **菁英共謀**
   - Keyword: 美國和台灣菁英共謀，剝削台灣人民
   - Tag: `共謀論`

6. **美國民主是假民主**
   - Keyword: 美國內部腐敗、美國的民主是假民主
   - Tag: `假民主`

7. **世界各國反美，美國人反對**
   - Keyword: 美國霸權遭受世界各國與美國人民反對
   - Tag: `反世界`

8. **戰場，毀滅台灣**
   - Keyword: 美國把台灣變成戰場，導致毀滅
   - Tag: `毀台論`

---

## Step 6：附加標籤

- 若文章同時符合多個面向，則可以同時標註多個 tag，例如：
  - `資訊操作` + `謬誤:滑坡` + `棄子論` + `反美`

---

# 最終輸出格式（範例）

輸出為一組 JSON Tag 陣列，例：

```json
{
  "tags": [
    "資訊操作",
    "來源:中國",
    "事實有誤",
    "謬誤:滑坡",
    "謬誤:不當類比",
    "棄子論",
    "戰場論"
  ],
  "comments": "此文章來自中國資訊來源，內容存在事實錯誤，並透過滑坡謬誤與不當類比誤導讀者。例如文中聲稱『美國最終會拋棄台灣，使台灣成為戰場』，這屬於典型的棄子論與戰場論論述，顯示其帶有資訊操作的傾向。"
}
```
